DILIP PORTFOLIO — deploy to GitHub Pages
1. Upload index.html AND the models/ folder (keep structure) to your repo.
2. Settings -> Pages -> deploy from branch (main /root).
3. Open the Pages URL. The 3D loads all 6 models; tab shows "DILIP Portfolio".
Note: the in-chat preview shows 4 models inline; the heavy machine + trailer load
from models/ once hosted.
